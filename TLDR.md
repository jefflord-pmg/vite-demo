# Vite Demo - TLDR

## What It Does
A Multi-Page Application (MPA) demonstrating modern Vite tooling with legacy libraries:
- **jQuery 3.7.1** + **Bootstrap 3.4.1**
- **TypeScript** for all code
- **Less** for CSS preprocessing
- **vite-plugin-checker** for real-time TypeScript errors
- Custom plugin for `.aspx` file watching

Two demo pages:
- **Hello World** - Counter, jQuery alert, Christmas countdown
- **CRUD Testing** - Add/edit/delete items demo
